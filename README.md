# PerfectSky Post
Web App  
https://bunesky.github.io/perfectsky-post/

PerfectSky Post was the first prototype analyzing posts from the Trending Feed.  
It provides basic metrics:

- Character and word averages  
- Media usage  
- Post type distribution  
- Hashtag frequency
